# Desafio 04 — Sistema de Loja Simples

## Descrição
Programa que simula uma loja, permitindo cadastro de produtos,
realização de vendas e geração de relatórios.

## Estrutura
Desafio 04/
├── loja/
│   ├── produtos.py
│   ├── vendas.py
│   └── relatorio.py
└── main.py

## Funcionalidades
- Cadastrar produtos
- Realizar vendas com cálculo de desconto
- Gerar relatório de vendas
- Salvar relatório em arquivo .txt

## Como rodar
Execute o arquivo `main.py` e siga o menu interativo.
